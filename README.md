# 在线订餐管理系统

## 1、项目介绍

在线订餐管理系统拥有两种角色

管理员：菜品管理、类别管理、用户管理、订单管理、评价用户、留言管理等

用户：登录注册、点餐、购物车、历史订餐、留言


## 2、项目技术

后端框架： Servlet、mvc模式

前端技术：jsp、css、JavaScript、JQuery

## 3、开发环境

- JAVA版本：JDK1.8
- IDE类型：IDEA、Eclipse都可运行
- tomcat版本：Tomcat 7-10版本均可
- 数据库类型：MySql（5.x和8.x版本都可） 
- maven项目：否
- 硬件环境：Windows 或者 Mac OS
